Project Pool

拟开展项目

遥感 + 灾害

基于遥感影像的地质灾害隐患识别

利用遥感影像和深度学习方法识别滑坡、泥石流等潜在隐患区域。

技术路线
影像预处理、样本标注、语义分割、风险可视化。
适合学生
算法、遥感 GIS、前端可视化方向。
无人机 + 识别

无人机图像边坡裂缝检测

面向边坡巡检图像，构建裂缝目标检测与巡检报告生成原型。

技术路线
图像采集、目标标注、YOLO 训练、结果展示。
适合学生
图像识别、模型训练、演示系统开发方向。
三维 + Web

三维地质模型 Web 展示系统

将钻孔、地层和剖面数据转化为可交互的三维地质展示页面。

技术路线
数据整理、三维建模、Three.js、Web 交互。
适合学生
前端开发、三维可视化、地质建模方向。
数据 + 找矿

地质找矿数据智能分析系统

围绕地球化学和空间数据，构建异常识别、靶区推荐和图表分析工具。

技术路线
数据清洗、特征工程、机器学习、地图展示。
适合学生
数据分析、机器学习、GIS 应用方向。
